# Basic

Pythonの基本的なこと。

## help

	$ python
	>>> help()		<=Intaractive mode
	help> divmod	<=調べたいものを入力する
	help> q			<=終了するには'q'またはEnterをタイプする
	>>> help(divmod) <=調べたいものを引数で指定する
